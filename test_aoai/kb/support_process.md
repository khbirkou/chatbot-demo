# Support Process

## Intake
1) Create a Work Order
2) Assign owner
3) Set priority (HIGH/MEDIUM/LOW)
4) Update mower status

## Work Order Lifecycle
OPEN → IN_PROGRESS → WAITING_PARTS → DONE

## Communication
- Customer updates only after status changes (OPEN, IN_PROGRESS, DONE).
- If WAITING_PARTS lasts > 3 days → notify Service Hub.
